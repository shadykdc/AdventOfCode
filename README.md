# Advent Of Code

[https://adventofcode.com/](https://adventofcode.com/)


## 2020

4/25 days completed in C++.

To run (on a mac at least):

1. Clone the repository.
2. Update the makefile with the day you would like to run. e.g. `DAY = 1`
3. Run `make`.
4. Run `./aoc`.
